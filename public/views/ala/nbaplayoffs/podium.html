The game structure for the original edition of the NBA Playoffs Pool was simple and traditional: a standard bracket game with correct playoff series picks doubling in value in each successive round. This was also fairly boring, and if this pool ever returns as an ALA game it will be in an enhanced format that includes a regular season component and additional tweaks to make the pool a little more interesting.
