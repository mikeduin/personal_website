A Note on 'Effective'/Normalized Rotisserie Scores
Due to the varying number of teams and Roto Points available in each season, included in each season's stats -- and incorporated into the Career Roto rankings -- are normalized Roto Scores displayed as "Effective Hitting," "Effective Pitching," and "Effective Total" scores. These metrics convert each Roto Score to an equivalent scale, scored from 0.67 - 10.0, which measures the percentage of Roto Points that a team amasses relative to the total number of Roto Points available in a given season. For example: if a team led the league in every hitting category in a given season, their effHit score for that season would be 10.0. If a team earned exactly half of all available hitting points, their effHit score would be 5.0; if a team got last in every hitting category, their effHit score would be 0.67 (it is 0.67 rather than 0.00 because last place in every category gets one point, so a team with the minimum number of points across all categories would still have 10 points). The scale works the same way for Pitching and Total Roto scores.

These numbers for each season are then weighed against the number of Roto Points available in the CURRENT season to give each manager a Best, Worst, and Average Roto Score in the 'Career Roto' section, and each manager's Average Score is then used as the basis for their Career Roto ranking. This normalized scale is necessary so that owners are not rewarded or penalized for playing in seasons in which there are more or less teams or roto points available than in other seasons. Each manager's Best, Worst, and Average Roto Scores should not be taken at face value, but instead interpreted as scores relative to the league's current scoring parameters (e.g., Joe Simich's best fantasy team could be expected to score 101 points in the current league format). - The Commish
